ZGMF-515 CGUE  
- Nom : ZGMF-515 CGUE.   
- Type : Mobile Suit Commander Type   
- Pilote : Rau Le Creuset   
- Créateur : ZAFT   
- Mecha Designer : Kunio Okawara.   
- Armement : MA-M4A Epée Lourde, MMI-M7S 76 mm mitrailleuse lourde stockée dans le dos M7070 Bouclier-plus-28 mm Vulcan.
